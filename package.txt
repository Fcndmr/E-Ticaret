backend için yapılacaklar

npm init
npm i

1- express : http protokolünü kullanarak işlem yapmak için --> npm install express --save
2- cors : api politikaları izinleri için --> npm i cors
3- nodemon : sunucuyu değişikliklerinden sonra otomatik olarak tekrar başlatmak için --> npm i -g nodemon (daha öce hiç kurulmadıysa yapılır)
4- dotenv : env dosyalarını tanımlamak için --> npm i dotenv
5- mongoose : mongo db üzerinde işlem yapabilmek için --> npm i mongoose
6- uuid : id üretmek için --> npm i uuid

.gitignore --> node js gitignore


server.js kurulduktan sonra node server.js --- nodemon server.js
